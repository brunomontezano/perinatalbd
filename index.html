Fatores perinatais e desenvolvimento de transtorno bipolar

Bruno Montezano

2022-06-25

Introduo

O presente trabalho almeja observar os efeitos de fatores perinatais e do aleitamento materno no desenvolvimento de transtorno bipolar (TB) aos 22 anos de idade. Os dados utilizados para tal foram coletados em uma coorte de nascimento coordenada pelo Centro de Epidemiologia da Universidade Federal de Pelotas intitulada Coorte de 1993.

# Carrega dados da Coorte de 1993
dados_coorte <- haven::read_sav("data/banco_QI_15_07.sav")

Transtorno bipolar

O diagnstico de transtorno bipolar ser calculado pela MINI (Mini International Neuropsychiatric Interview) por meio dos critrios diagnsticos do DSM-IV. Sero calculados diagnsticos de transtorno bipolar tipo I e transtorno bipolar tipo II.

Construo de variveis e tcnicas de imputao

As seguintes variveis foram construdas:

  • tb_dic: Diagnstico de TB tipo I ou TB tipo II (dicotmica)
  • cesarea: O parto foi cesria?
  • parto_induzido: A bolsa rompeu, foi utilizado soro ou ambas as coisas?
  • uti_neo: O recm-nascido foi para UTI?
  • complicacoes: Variveis de prematuridade, traumatismo no parto, disfuno respiratria no parto, malformao no parto e sofrimento fetal agrupadas (atravs do operador OU)
  • hipert_materna: Hipertenso arterial materna
  • dm_materna: Diabetes materna
  • infec_gest: Infeco gestacional
  • anemia_gest: Anemia gestacional
  • hemorr_parto: Apresentou hemorragia no parto
  • tab_gest: Contato com tabagismo na gestao (ativo ou passivo)
  • etil_pesado_gest: Etilismo pesado na gestao
  • rup_famil: Ruptura familiar
  • baixo_peso: Baixo peso ao nascer
  • baixo_comp: Baixo comprimento ao nascer
  • per_cef: Permetro ceflico (numrico)
  • escol_paterna: Escolaridade paterna
  • escol_materna: Escolaridade materna
  • paridade: Paridade (dicotmica) [4 ou mais e menos que 4]
  • apoio_familiar: Teve apoio da famla/vizinhos/amigos?
  • idade_mae_num: Idade da me (numrica)
  • idade_mae_20: Idade da me (< 20 anos)
  • idade_mae_34: Idade da me (< 34 anos)
  • pretende_amamentar: Me pretende amamentar
  • apgar_1: Apgar no minuto 1 (escore bruto)
  • apgar_5: Apgar no minuto 5 (escore bruto)
  • renda_familiar: Renda familiar (em salrios mnimos)

As variveis de idade materna foram utilizadas atravs das trs codificaes pois os autores consideraram a hiptese da relao da idade com o desfecho no ser puramente linear. Dessa forma, esta foi uma forma na qual se pensou uma maneira de considerar possvel associao.

da_proc <- da |>
    dplyr::transmute(
        # Desfecho: transtorno bipolar aos 22 anos
        tb_dic = factor(dplyr::if_else(TB_final == 1,
                                         "Sim",
                                         "No"
                                         ),
                        levels = c("No", "Sim")
                           ),
        # Cesrea ou parto normal
        cesarea = as.factor(dplyr::if_else(apartip == "normal",
                                           "No",
                                           "Sim"
                                           )
                            ),
        # Parto induzido
        parto_induzido = as.factor(dplyr::if_else(
            ainduz %in% c("rompeu bolsa", "soro", "ambos"),
            "Sim",
            "No"
        )),
        # Ida a UTI
        uti_neo = as.factor(dplyr::if_else(
            aprobrn %in% c("bercario", "aloj.conjunto"),
            "No",
            dplyr::if_else(aprobrn == "uti", "Sim",
                           NA_character_
                           )
        )),
        # Nascimento prematuro
        prematuro = as.factor(
            dplyr::if_else(aerrado != "prematuro" | is.na(aerrado),
                           "No",
                           "Sim"
                           )
        ),
        # Traumatismo no parto
        traumatismo = as.factor(
            dplyr::if_else(
                aprobrn1 == "traumatismo" | aprobrn2 == "traumatismo" |
                    aprobrn3 == "traumatismo",
                "Sim",
                "No",
                missing = "No"
            )
        ),
        disf_resp = as.factor(
            dplyr::if_else(
                aprobrn1 == "stress resp" | aprobrn2 == "stress resp" |
                    aprobrn3 == "stress resp",
                "Sim",
                "No",
                missing = "No"
            )
        ),
        malform = as.factor(
            dplyr::if_else(
                aprobrn1 == "malformacao" | aprobrn2 == "malformacao" |
                    aprobrn3 == "malformacao",
                "Sim",
                "No",
                missing = "No"
            )
        ),
        sofrfetal = as.factor(
            dplyr::if_else(
                aprobrn1 == "sofrim fetal" | aprobrn2 == "sofrim fetal" |
                    aprobrn3 == "sofrim fetal",
                "Sim",
                "No",
                missing = "No"
            )
        ),
        hipert_materna = relevel(ahipert, ref = "nao"),
        dm_materna = as.factor(dplyr::if_else(
            adiabet %in% c("tratado", "nao tratado"),
            "Sim",
            "No"
        )),
        infec_gest = as.factor(
            dplyr::if_else(
                ainfecur %in% c("tratado", "nao tratado") |
                    aoutinf %in% c("tratado", "nao tratado"),
                "Sim",
                "No"
            )
        ),
        anemia_gest = as.factor(dplyr::if_else(
            aanemia %in% c("tratado", "nao tratado"),
            "Sim",
            "No"
        )),
        hemorr_parto = as.factor(
            dplyr::if_else(
                amotcesa == "hemorragia materna" | apartind == "sangramento",
                "Sim",
                "No"
            )
        ),
        tab_gest = as.factor(
            dplyr::if_else(afumou == "sim" | acompfum == "sim",
                           "Sim",
                           "No"
                           )
        ),
        etil_pesado_gest = as.factor(
            dplyr::if_else(abebalc == "sim",
                           "Sim",
                           "No"
                           )
        ),
        rup_famil = as.factor(
            dplyr::if_else(avivmar == "sim",
                           "No",
                           "Sim"
                           )
        ),
       baixo_peso = as.factor(
           dplyr::if_else(apesorn < 2500,
                          "Sim",
                          "No"
                          )
       ),
       baixo_comp = as.factor(
           dplyr::if_else(acompr < 47,
                          "Sim",
                          "No"
                          )
       ),
       per_cef = apcrn,
       per_cef_curto = as.factor(
           dplyr::if_else(apcrn < 33,
                          "Sim",
                          "No"
                          )
       ),
       escol_paterna = aescpai,
       escol_materna = aescmae,
       paridade = relevel(as.factor(dplyr::if_else(
           aparidad %in% c("0", "1", "2", "3"),
            "Menor que 4",
            "Maior ou igual a 4")
           ), ref = "Menor que 4"), 
       apoio_familiar = as.factor(
           dplyr::if_else(
               aapoio == "sim",
               "Sim",
               "No"
           )
       ),
       idade_mae_num = aidadmae,
       idade_mae_20 = as.factor(
           dplyr::if_else(
               aidadmae < 20,
               "Menor que 20",
               "Maior ou igual a 20"
           )
       ),
       idade_mae_34 = as.factor(
           dplyr::if_else(
               aidadmae < 34,
               "Menor que 34",
               "Maior ou igual que 34"
           )
       ),
       pretende_amamentar = as.factor(
           dplyr::if_else(
               apretama == "sim",
               "Sim",
               "No"
           )
       ),
       apgar_1 = aapgar1,
       apgar_5 = aapgar5,
       renda_familiar = arenfam
    )

# Agrupar variveis de complicaes perinatais
da_proc <- da_proc |> 
    dplyr::mutate(
        complicacoes = as.factor(
            dplyr::if_else(
                prematuro == "Sim" | traumatismo == "Sim" |
                disf_resp == "Sim" | malform == "Sim" |
                sofrfetal == "Sim",
                "Sim", "No"
            )
        ),
        .keep = "unused"
    )

# Criar base com variveis imputadas pela moda e mediana
da_imp <- da_proc |> 
    recipes::recipe(tb_dic ~ .) |> 
    recipes::step_filter_missing(recipes::all_predictors(), threshold = 0.1) |> 
    recipes::step_impute_mode(recipes::all_nominal_predictors()) |> 
    recipes::step_impute_median(recipes::all_numeric_predictors()) |> 
    recipes::step_nzv(recipes::all_predictors()) |> 
    recipes::prep() |> 
    recipes::bake(new_data = NULL)

Anlise multivariada

A imputao foi realizada nos passos anteriores para a anlise bivariada, logo, o objeto da base de dados imputada (pr-processada) ser usada para o ajuste do modelo de regresso logstica binomial.

Um modelo contendo somente as variveis com valor p menor que 0,2 na anlise bivariada foi construdo.

Os valores de odds ratio foram estimados a partir da equao \(e^{\beta}\).

Modelo de regresso logstica binomial

Neste modelo, foram includas somente as variveis independentes que apresentaram um valor de p menor que 0,2 na anlise bivariada (bruta).

vars_multivar <- cat_biv |>
